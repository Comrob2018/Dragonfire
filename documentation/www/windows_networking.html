Basic Networking

A variety of commands can be used to gather information about the box you are currently on.

List network interfaces on the box
ipconfig /all
View ARP cache
arp -a
Ping via Powershell
test-Connection [IP]
Add a static IP address to an interface
netsh interface ip set address name="Local Area Connection" static [ip] [netmask]] [gateway]

Firewall

netsh can be used to perform queries and manage the firewall.


View firewall config (XP+)
netsh firewall show config
View firewall Rules (Vista+)
netsh advfirewall firewall show rule name=all
View Current Profile (Vista+)
netsh advfirewall show currentprofile
Enable Remote Desktop (Vista+)
netsh advfirewall firewall set rule group="remote desktop" new enable=Yes
Open a specific port (Vista+)
netsh advfirewall firewall add rule name="allow telnet" dir=in action=allow protocol=TCP localport=23
Delete a rule (Vista+)
netsh advfirewall firewall delete rule name="allow telnet"
Disable logging of dropped packets (Vista+)
netsh firewall set logging droppedpackets=disable

Tunneling

netsh can be used to create a port relay through a Windows box.


Port relay through a box to another box (Win7 or later supports v4tov4, v6tov6, v4tov6 and v6tov4)
netsh interface portproxy add v4tov4 listenport=[LPORT] listenaddress=0.0.0.0 connectport=[RPORT] connectaddress=[RHOST]

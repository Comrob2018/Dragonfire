Basic User Management

net use can be used to perform queries and manage users.

List users on machine
net user
View details about user
net user [user]
Add user
net user [user] [password] /add
Delete user
net user [user] /delete

Powershell can be used to perform queries and manage users. The below commands work on PS 2.0 (WIn7)


List users and details on machine
Get-WmiObject -Class Win32_UserAccount

Basic Group Management

net use can be used to perform queries and manage groups.


List groups on machine
net localgroup
Add user to administrators group
net localgroup administrators [user] /add

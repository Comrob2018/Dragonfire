Dragonfire

Welcome to the DragonFire toolkit library. DragonFire is a collection of open-source tools, techniques, and scripts. This documentations accompanies the DragonFire tarball.


DragonFire isn't all encompasing of every possible techniqe, exploit, process, etc. Rather it is good documentation to use as a starting point. If there is a technique or exploit not documented here, test it, make sure you know what you're doing, and go for it. Better yet, get it added to DragonFire.

Installation

DragonFire is designed to be used with a Kali machine. There are binaries from Kali that DragonFire relies on. To being, just extract the DragonFire tarball into the /dragonfire directory of your Kali machine. In the /dragonfire folder there is a file called sourceme that will set your path and add some aliases. Run source /dragonfire/sourceme in each terminal, or add the line to your .bashrc file.
mkdir /dragonfire
tar -xvf dragonfire.tar -C /dragonfire

Once aliases are set, get-documentation (alias=gd) will launch the help files. Running gd windows will open the documentation to the windows page. This will work for any of the four major pages. (Linux, Windows, Other, Misc)

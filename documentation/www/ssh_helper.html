SSH Helper

This script makes SSH Multiplexing easier, it is not a replacement for reading and understanding the main SSH Usage page.

Use this helper script to quickly create an SSH Multiplex session, upload and download files easily, and manage tunnels. You can start the script BEFORE you have your multiplex session created, it will make pastables for you. Using the script is no different than running commands manually, as detailed on the SSH Usage page, however it's useful anytime you have to create tunnels, upload or download many files, or require several terminal windows.


Additionally, this script will print each command it runs before it executes so you can verify the command and copy it for later if needed.

Usage

Usage: Running with no arguments will prompt for entry. There is no prompt for port, you must enter it as an argument if it is not 22.
python /dragonfire/bin/ssh_helper/ssh_helper.py

-u : username

-p : port (default 22)

-t : target address

-h : help

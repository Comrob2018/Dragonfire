SSH Survey Script

Basic situational awareness script when connecting via SSH. Uses Multiplexing so no additional authenications are required and stdout can be redirected locally.

Information

Work With: Linux
Requires: ssh service, credentials

SSH Survey Script

After connecting to a target via SSH, the SSH Survey Script will gather valuable situational awareness information quickly, without additional authenication on target. Pastable line for the SSH Survey will be created for you if you are using the ssh_helper to create your ssh sessions.
Recommend redirecting stdout to a local file and then viewing the output.

SSH Survey does not check EVERYTHING you might want to see. It not running a command in no excuse for poor situational awareness.


Usage

/dragonfire/bin/ssh_survey/ssh_survey.sh [USER] [IP] [CONTROL_FILE]
This will run the survey and dump data all over the screen. You should probably redirect this output. Full details of everything that run, as well as their headers (so you can search) are below.
